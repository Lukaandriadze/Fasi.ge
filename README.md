
# Fasi.ge 

**Fasi.ge** არის პროდუქტის ფასების შედარების პლატფორმა, რომელიც მომხმარებელს ეხმარება სხვადასხვა ქართული ონლაინ მაღაზიებიდან სასურველი პროდუქტის საუკეთესო ფასის სწრაფად პოვნაში.

პლატფორმა აგროვებს ინფორმაციას სხვადასხვა მაღაზიებიდან, აერთიანებს პროდუქტებს ერთ სივრცეში და მომხმარებელს აჩვენებს ფასებს, მახასიათებლებს, მაღაზიას და პირდაპირ ბმულს პროდუქტის გვერდზე.

# პროექტის იდეა

ონლაინ შოპინგის დროს მომხმარებელს ხშირად უწევს სხვადასხვა ვებსაიტზე ცალ-ცალკე შესვლა და ფასების შედარება. ეს პროცესი დიდ დროს მოითხოვს და რთულია ყველა არსებული შეთავაზების მოძიება.

Fasi.ge ამ პრობლემას აგვარებს ერთი საერთო საძიებო სისტემის შექმნით, სადაც მომხმარებელს შეუძლია მოძებნოს პროდუქტი და ერთ ადგილას ნახოს სხვადასხვა მაღაზიაში არსებული ვარიანტები.
